# Sticky media

You may add a sticky video by giving `sticky-popup` class to an `a` tag.

```html
<!-- md content above -->

<a
    // highlight-next-line
    class="sticky-popup"
    target="_blank"
    href="https://www.youtube.com/watch?v=G1OM2L7XK5Y"
    >Sticky video</a
>

<!-- md content below -->
```
